## Canva Groups

1. [Download CSV from Supergroups directory](https://canva.world/supergroups/directory) and save `canvaworld-supergroups.csv` to `/data`
2. Copy table to a [new Google Doc](https://sheets.new) and export as Markdown, saving as `canvaworld-supergroups.md` in `/data`
3. `bun run build` + `bun preview` or just `bun dev`
